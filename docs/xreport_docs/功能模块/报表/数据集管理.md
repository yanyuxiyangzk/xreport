# 数据集管理

## 概述

数据集（Dataset）是报表中绑定到单元格的 SQL 查询结果。通过数据集，报表单元格可以动态显示数据库中的数据。

## 数据集与数据源的关系

```
数据源 ──── 多个 ────→ 数据集
（数据库连接）         （具体SQL查询）
```

一个数据源可以创建多个数据集，每个数据集对应一条 SQL 查询。

## 创建数据集

### 入口

在报表设计器右侧面板，点击「数据集」选项卡，点击「添加数据集」。

### 步骤一：选择数据源

从已配置的数据源列表中选择一个。

### 步骤二：编写 SQL

```sql
SELECT
    dept_name   AS "部门名称",
    user_count  AS "用户数量",
    create_time AS "创建时间"
FROM sys_dept
WHERE 1=1
    AND status = 1
ORDER BY user_count DESC
```

### 步骤三：设置数据集名称

系统自动生成编号 `ds001`，可自定义名称便于在单元格公式中引用。

### 步骤四：执行预览

点击「执行」按钮，预览查询结果。确认数据无误后保存。

## 动态参数

### 参数语法

在 SQL 中使用 `${paramName}` 引用参数：

```sql
SELECT * FROM sys_user
WHERE 1=1
  AND username LIKE '%${keyword}%'
  AND dept_id = ${deptId}
```

### 参数与控件绑定

参数定义后，需要在「参数配置」中为每个参数绑定一个控件（输入框、下拉框、日期选择等）。

### 参数类型

| SQL 写法 | 说明 | 对应控件 |
|----------|------|----------|
| `${name}` | 字符串参数 | 字符串输入框 |
| `${age}` | 数字参数 | 数字输入框 |
| `${startDate}` | 日期参数 | 日期选择器 |
| `${status:1}` | 带默认值的参数 | 支持默认值 |

### 日期范围参数

日期范围参数会拆分为两个参数：

```sql
-- ${startDate} 和 ${endDate}
SELECT * FROM orders
WHERE create_time BETWEEN '${startDate}' AND '${endDate}'
```

## 数据集类型

### 普通数据集

最常用的类型，一条 SQL 返回一个结果集。

### 主子表数据集

当报表需要展示主表和从表的多层数据时使用，如订单主表 + 订单明细从表：

```sql
-- 主表
SELECT order_id, customer_name, total_amount
FROM orders
WHERE order_id = ${orderId}

-- 从表（通过 ${orderId} 过滤）
SELECT product_name, quantity, price
FROM order_items
WHERE order_id = ${orderId}
```

### 公共数据集（规划中）

公共数据集可供多个报表复用，避免重复编写相同的 SQL。

## SQL 编写规范

### 基本规范

1. **总是加 `1=1`**：便于后续 AND 条件追加
2. **使用别名**：为每个字段指定 AS 别名，中文别名更易读
3. **避免 SELECT ***：明确列出需要的字段，减少数据传输
4. **注意 SQL 注入**：参数化查询，禁止字符串拼接

### 分页查询

当数据量可能很大时，启用分页：

```sql
SELECT * FROM (
    SELECT t.*, ROW_NUMBER() OVER (ORDER BY id) AS row_num
    FROM big_table t
    WHERE 1=1 ${condition}
) AS t2
WHERE row_num BETWEEN ${offset} AND ${offset + pageSize}
```

### 性能优化

- 为 WHERE 条件的字段建立索引
- 避免在 WHERE 中对字段使用函数（如 `DATE(create_time)`）
- 关联查询注意 JOIN 顺序，小表放左

## 数据集函数

### ds.xxx 引用

在单元格中使用以下公式引用数据：

| 公式 | 说明 | 示例 |
|------|------|------|
| `=${ds001.field}` | 引用单个字段 | `=${ds001.username}` |
| `=${ds001.*}` | 引用整行 | 配合 `ROW()` 使用 |

### 常用 Excel 函数组合

配合 Excel 原生函数实现复杂效果：

```
=SUM(ds001.amount)           -- 求和
=AVERAGE(ds001.score)        -- 平均值
=MAX(ds001.value)            -- 最大值
=COUNTIF(ds001.status,"1")   -- 条件计数
```

### 动态序号

配合 `ROW()` 实现自动序号：

```sql
-- 在 SQL 中添加序列列
SELECT *, ROW_NUMBER() OVER (ORDER BY id) AS seq FROM table_name
```

单元格公式：`=${ds001.seq}`
