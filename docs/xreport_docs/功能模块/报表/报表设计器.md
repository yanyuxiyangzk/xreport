# 报表设计器

## 概述

XReport 报表设计器是基于 Luckysheet 实现的在线表格设计工具，类 Excel 操作体验，拖拽式报表设计，无需写代码，几分钟即可完成一张报表。

## 核心能力

- **类 Excel 操作**：复制、粘贴、插入行列、合并单元格等
- **拖拽设计**：直接拖拽字段到单元格
- **多 Sheet 支持**：一个报表模板可包含多个 Sheet
- **参数查询**：支持多种参数组件，灵活筛选数据
- **图表集成**：内置 Chart.js，支持多种图表类型
- **条件格式**：支持单元格背景色、字体颜色等条件格式

## 操作路径

菜单路径：`报表管理 → 报表设计`

## 创建报表

### 步骤一：新建报表

1. 点击「新建报表」按钮
2. 填写报表名称、报表编码
3. 选择所属分类（可选）
4. 点击「确定」进入设计器

### 步骤二：设计表格

在 Luckysheet 设计器中：

- **设置表头**：在第一行输入列名称
- **合并单元格**：选中多个单元格，点击「合并居中」
- **设置边框**：选择单元格范围，设置边框样式
- **调整列宽行高**：拖拽行号列或列标题行

### 步骤三：绑定数据

1. 在右侧「数据集」面板，点击「添加数据集」
2. 选择已配置的数据源
3. 编写 SQL 查询语句
4. 点击「执行」预览数据

### 步骤四：绑定字段

在需要显示数据的单元格中，使用公式引用数据集字段：

```
=${ds001.field_name}
```

其中 `ds001` 为数据集编号，`field_name` 为字段名。

### 步骤五：配置参数（可选）

点击「参数配置」，添加查询条件：

| 参数类型 | 说明 |
|----------|------|
| 字符串输入框 | 单行文本输入 |
| 数字输入框 | 数值输入 |
| 下拉单选 | 从预置列表选择一项 |
| 下拉多选 | 从预置列表选择多项 |
| 日期选择 | 单个日期 |
| 日期范围 | 起止日期范围 |
| 级联下拉 | 多级联动选择 |

参数使用方式：在 SQL 中通过 `${paramName}` 引用。

### 步骤六：保存预览

点击「保存」保存报表模板，点击「预览」查看实际效果。

## 图表设计

### 支持图表类型

- 柱状图（Bar）
- 折线图（Line）
- 饼图（Pie）
- 散点图（Scatter）
- 面积图（Area）

### 插入图表步骤

1. 选中要放置图表的起始单元格
2. 点击工具栏「添加图表」
3. 选择图表类型
4. 绑定 X 轴字段和 Y 轴字段
5. 调整图表大小和位置

## 单元格属性

### 数据填充方式

| 填充方式 | 说明 |
|----------|------|
| 普通填充 | 数据直接填入 |
| 横向扩展 | 数据从左到右扩展 |
| 纵向扩展 | 数据从上到下扩展（默认） |

### 数据格式化

支持对数字、日期、货币等设置格式化模板，如 `#,##0.00`（千分位，保留两位小数）。

## 报表导出

报表设计完成后，支持三种导出格式：

| 格式 | 说明 |
|------|------|
| Excel (.xlsx) | 保留样式和数据 |
| PDF | 适合打印存档 |
| CSV | 纯数据，便于数据再处理 |

导出时可以指定是否包含参数查询区域。

## 最佳实践

1. **设计前先规划**：先设计纸质报表原型，再在设计器中实现
2. **合理使用合并单元格**：合并单元格影响数据扩展，谨慎使用
3. **SQL 优化**：数据量大时注意 SQL 性能，避免全表扫描
4. **参数使用**：复杂条件用参数，固定条件直接写在 SQL 中
5. **多 Sheet 协作**：相关但不同的数据用多个 Sheet 展示
