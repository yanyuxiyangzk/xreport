# 常见问题

## 1. XReport 是否完全开源？

是的，XReport 是完全开源项目，遵循 MIT License 协议，可免费商用。

源码地址：https://github.com/yanyuxiyangzk/xreport

## 2. 支持哪些数据库？

当前版本支持以下数据库：
- MySQL 5.7+
- PostgreSQL 12+

其他数据库（如 Oracle、SQL Server、达梦等）已在规划中。

## 3. 前端支持 Vue 吗？

当前版本前端使用 **React 18 + TypeScript + Vite**。Vue 版本暂无计划。

## 4. SQL 语句中小于号（<）如何处理？

由于 MyBatis XML 中 `<` 是标签的开始符号，直接使用小于号会报错。有三种解决方式：

**方式一：使用 HTML 实体**
```sql
SELECT * FROM table WHERE 字段A &lt; 字段B
```

**方式二：交换比较顺序**
```sql
SELECT * FROM table WHERE 字段B > 字段A
```

**方式三：使用 CDATA 标签**
```xml
<![CDATA[SELECT * FROM table WHERE 字段A < 字段B]]>
```

## 5. 报表参数如何通过 URL 传递？

假设参数名称为 `menuName`，参数编码为 `menuName`，访问报表时在 URL 中带入参数：

```
http://localhost:8080/xreport/report/preview?tplId=123456&menuName=test
```

参数值会自动填充到查询框中，并作为查询条件执行。

## 6. 如何配置数据源？

进入 **数据源管理** 页面，填写以下信息：

| 字段 | 说明 |
|------|------|
| 数据源名称 | 标识名称，如"生产库" |
| 数据库类型 | MySQL / PostgreSQL |
| 连接地址 | JDBC URL，如 `jdbc:mysql://localhost:3306/dbname` |
| 用户名 | 数据库用户名 |
| 密码 | 数据库密码 |

## 7. 导出格式有哪些？

当前支持以下三种导出格式：
- **Excel（.xlsx）**：保留原表格样式
- **PDF**：适合打印存档
- **CSV**：适合数据再处理

## 8. 如何设计一张报表？

设计报表的基本步骤：

1. **新建报表**：在报表列表页面点击"新建报表"
2. **设计表格**：拖拽单元格，设置表头和数据区域
3. **绑定数据源**：选择数据源，编写 SQL 查询语句
4. **配置参数**（可选）：添加查询条件，如日期范围、部门选择等
5. **添加图表**（可选）：插入柱状图、折线图等图表
6. **预览保存**：在线预览效果，确认无误后保存

## 9. 序号如何动态扩展？

可以将 `row()` 函数与动态数据集结合使用。例如在 SQL 中添加一列：

```sql
SELECT *, 0 AS sequence FROM table_name
```

单元格中写入公式：
```
=ROW()-ds001.${sequence}
```

这样序号列会随数据行数动态扩展。

## 10. 多表头如何设计？

在报表设计器中，通过合并单元格的方式实现多表头，并在需要的位置绑定对应的数据集字段。

## 11. 报表中如何使用图表？

在报表设计器的工具栏中，选择"插入图表"，然后：
1. 选择图表类型（柱状图、折线图、饼图等）
2. 绑定 X 轴和 Y 轴的数据字段
3. 调整图表位置和大小
