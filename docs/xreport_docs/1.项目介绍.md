# XReport 项目介绍

## 简介

**XReport** 是一款面向开发者和企业的在线报表平台，支持拖拽式 Excel 报表设计、数据源管理、报表执行与多格式导出（PDF / Excel / CSV）。

> 官网：https://xreport.vip（待上线）
> 源码：https://github.com/yanyuxiyangzk/xreport

## 核心特性

- **拖拽式报表设计**：类 Excel 风格，通过拖拽方式快速设计报表，无需写代码
- **多数据源支持**：支持 MySQL、PostgreSQL 等主流关系型数据库（JDBC）
- **丰富的图表支持**：集成 Chart.js，支持柱状图、折线图、饼图、散点图等多种图表
- **多格式导出**：支持 PDF、Excel、CSV 三种导出方式
- **参数查询**：支持字符串、数字、下拉、日期等多种查询参数
- **在线预览**：设计完成后可直接在线预览效果
- **开源友好**：MIT License，开源版本可免费商用

## 技术架构

| 层级 | 技术选型 |
|------|----------|
| 前端框架 | React 18 + TypeScript + Vite |
| UI 组件库 | Ant Design 5 |
| 表格引擎 | Luckysheet |
| 图表引擎 | Chart.js |
| 后端框架 | Spring Boot 3.2.6 + JDK 17 |
| ORM | MyBatis-Plus |
| 安全 | Spring Security + JWT |
| 数据库 | MySQL 5.7+ / PostgreSQL |

## 项目模块

XReport 采用 Maven 多模块结构：

```
xreport-parent/
├── xreport-common          # 公共模块（工具类、响应封装、异常处理）
├── xreport-pojo             # 实体类、数据传输对象
├── xreport-data            # 数据访问层（Mapper、SQL脚本）
├── xreport-service         # 业务逻辑层
└── xreport-web             # Web 层（Controller、Security 配置）
```

| 大屏设计 | 支持 | 规划中 |

| 多租户 | 支持 | 规划中 |

XReport 当前版本专注于 **Excel 类报表** 的设计与执行，聚焦核心功能，力图在单点做到极致。
