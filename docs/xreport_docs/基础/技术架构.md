# 技术架构

## 整体架构

XReport 采用经典的前后端分离架构：

```
┌─────────────────────────────────────────────────────┐
│                    用户浏览器                        │
└──────────────────┬────────────────────────────────┘
                   │ HTTP / REST API
┌──────────────────▼────────────────────────────────┐
│              XReport Web (Spring Boot)             │
│  ┌──────────┐  ┌──────────┐  ┌──────────────────┐ │
│  │Security  │  │Controller│  │  Service Layer   │ │
│  │JWT Filter│  │ REST API │  │  Business Logic  │ │
│  └──────────┘  └──────────┘  └──────────────────┘ │
│  ┌──────────────────────────────────────────────┐ │
│  │            MyBatis-Plus ORM                    │ │
│  └──────────────────────────────────────────────┘ │
└──────────────────┬────────────────────────────────┘
                   │ JDBC
     ┌─────────────┼─────────────────┐
     │             │                 │
┌────▼────┐  ┌────▼────┐  ┌────────▼──────┐
│  MySQL  │  │PostgreSQL│  │  Other DBs     │
│  5.7+   │  │  12+     │  │  (Planning)    │
└─────────┘  └─────────┘  └───────────────┘

┌─────────────────────────────────────────────────────┐
│           XReport Web UI (React 18)                 │
│  ┌──────────┐  ┌──────────┐  ┌──────────────────┐ │
│  │Router    │  │Ant Design│  │  Luckysheet       │ │
│  │(React    │  │ 5.x UI   │  │  (Spreadsheet)    │ │
│  │Router)   │  │          │  │                   │ │
│  └──────────┘  └──────────┘  └──────────────────┘ │
└─────────────────────────────────────────────────────┘
```

## 前端技术栈

| 技术 | 用途 |
|------|------|
| React 18 | 核心框架 |
| TypeScript | 类型安全 |
| Vite | 构建工具 |
| Ant Design 5 | UI 组件库 |
| Luckysheet | 在线表格编辑器 |
| Chart.js | 图表渲染 |
| Axios | HTTP 客户端 |
| React Router | 前端路由 |

## 后端技术栈

| 技术 | 用途 |
|------|------|
| Spring Boot 3.2.6 | 核心框架 |
| JDK 17 | 运行环境 |
| Spring Security | 安全认证 |
| JWT | Token 认证 |
| MyBatis-Plus | ORM 框架 |
| Redis | 缓存（可选） |
| Lombok | 简化代码 |
| Hutool | 工具库 |

## 项目模块设计

```
xreport-parent (pom.xml)
 │
 ├── xreport-common         # 公共基础模块
 │    ├── annotation/      # 自定义注解
 │    ├── constant/        # 常量定义
 │    ├── result/           # 统一响应封装（Result）
 │    ├── exception/        # 全局异常处理
 │    └── utils/            # 工具类
 │
 ├── xreport-pojo           # 实体对象模块
 │    ├── entity/           # 数据库实体（SysUser, SysRole...）
 │    ├── dto/              # 数据传输对象
 │    └── vo/               # 视图对象
 │
 ├── xreport-data           # 数据访问模块
 │    ├── mapper/           # MyBatis Mapper 接口
 │    └── resources/
 │         └── mapper/      # XML SQL 映射文件
 │         └── db/          # SQL 初始化脚本
 │
 ├── xreport-service        # 业务逻辑模块
 │    └── 目录结构按模块划分
 │
 └── xreport-web            # Web 层模块
      ├── controller/       # REST Controller
      ├── security/         # Security 配置 + JWT 过滤器
      └── resources/
           └── application.yml
```

## 报表执行流程

```
用户打开报表
    │
    ▼
前端请求 ──→ 后端获取报表模板配置
    │              │
    │              ▼
    │         执行数据源 SQL
    │              │
    │              ▼
    │         渲染 Luckysheet / Chart.js
    │
    ▼
返回渲染结果
    │
    ├── 在线预览（HTML）
    ├── 导出 Excel（通过 POI / EasyExcel）
    ├── 导出 PDF（通过 iText）
    └── 导出 CSV（流式输出）
```

## 报表数据结构

核心数据模型：

| 实体 | 说明 |
|------|------|
| ReportTpl | 报表模板主表 |
| ReportSheet | 报表 Sheet 页 |
| ReportDataSource | 数据源配置 |
| ReportDataset | 数据集（SQL 查询） |
| ReportParam | 报表参数 |
| SysUser | 系统用户 |
| SysRole | 系统角色 |
| SysDept | 部门 |
| SysMenu | 菜单 |

## 安全性设计

- 所有 API 需要携带有效 JWT Token
- Token 通过 `Authorization: Bearer <token>` 头部传递
- Token 有效期：24 小时（可配置）
- 密码使用 BCrypt 加密存储
- SQL 使用参数化查询，防注入
