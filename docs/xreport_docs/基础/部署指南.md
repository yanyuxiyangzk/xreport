# 部署指南

## 环境要求

| 环境 | 版本要求 |
|------|----------|
| JDK | 17+ |
| Node.js | 18+ |
| MySQL | 5.7+ 或 PostgreSQL 12+ |
| Maven | 3.8+ |
| Windows / Linux | 均可 |

## 后端部署

### 1. 环境检查

```bash
java -version
# OpenJDK 17.x.x

mvn -version
# Apache Maven 3.8.x
```

### 2. 初始化数据库

创建数据库（MySQL 示例）：

```sql
CREATE DATABASE xreport DEFAULT CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
CREATE USER 'xreport'@'%' IDENTIFIED BY 'xreport123';
GRANT ALL PRIVILEGES ON xreport.* TO 'xreport'@'%';
FLUSH PRIVILEGES;
```

导入初始化脚本：

```bash
# 脚本位于 xreport-data/src/main/resources/db/ 目录
mysql -uxreport -pxreport123 xreport < xreport-init.sql
```

### 3. 修改配置

编辑 `xreport-web/src/main/resources/application.yml`：

```yaml
spring:
  datasource:
    url: jdbc:mysql://localhost:3306/xreport?useUnicode=true&characterEncoding=utf8&serverTimezone=Asia/Shanghai
    username: xreport
    password: xreport123
    driver-class-name: com.mysql.cj.jdbc.Driver

server:
  port: 8080
  servlet:
    context-path: /xreport

jwt:
  secret: your-256-bit-secret-key-here
  expiration: 86400000  # 24小时

xreport:
  upload:
    path: /data/xreport/uploads
```

### 4. 编译打包

```bash
mvn clean package -DskipTests
```

生成的 JAR 文件：`xreport-web/target/xreport-web.jar`

### 5. 启动服务

**开发环境（直接运行）：**
```bash
cd xreport-web
mvn spring-boot:run
```

**生产环境：**
```bash
java -jar xreport-web.jar --spring.profiles.active=prod
```

**后台运行（Linux）：**
```bash
nohup java -jar xreport-web.jar --spring.profiles.active=prod > xreport.log 2>&1 &
```

## 前端部署

### 1. 安装依赖

```bash
cd xreport-ui
npm install
```

### 2. 开发环境启动

```bash
npm run dev
# 访问 http://localhost:5173
```

### 3. 生产环境构建

```bash
npm run build
# 构建产物输出到 dist/ 目录
```

### 4. Nginx 配置

```nginx
server {
    listen 80;
    server_name your-domain.com;

    # 前端静态文件
    location / {
        root /var/www/xreport-ui/dist;
        try_files $uri $uri/ /index.html;
    }

    # 后端 API 代理
    location /xreport/ {
        proxy_pass http://127.0.0.1:8080/;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
    }

    # 上传文件代理
    location /xreport/uploads/ {
        alias /data/xreport/uploads/;
        expires 7d;
    }
}
```

## Docker 部署（推荐）

### 后端 Docker

```dockerfile
FROM eclipse-temurin:17-jre-alpine
COPY target/xreport-web.jar /app/app.jar
WORKDIR /app
EXPOSE 8080
ENTRYPOINT ["java", "-jar", "app.jar"]
```

```bash
docker build -t xreport-backend:latest .
docker run -d -p 8080:8080 \
  -e SPRING_DATASOURCE_URL=jdbc:mysql://mysql:3306/xreport \
  -e SPRING_DATASOURCE_USERNAME=xreport \
  -e SPRING_DATASOURCE_PASSWORD=xreport123 \
  --name xreport-backend \
  --link mysql \
  xreport-backend:latest
```

### Docker Compose 一键启动

```yaml
version: '3.8'
services:
  mysql:
    image: mysql:8.0
    environment:
      MYSQL_ROOT_PASSWORD: root123
      MYSQL_DATABASE: xreport
      MYSQL_USER: xreport
      MYSQL_PASSWORD: xreport123
    volumes:
      - mysql_data:/var/lib/mysql
      - ./db/xreport-init.sql:/docker-entrypoint-initdb.d/init.sql
    ports:
      - "3306:3306"

  xreport-backend:
    build: ./backend
    ports:
      - "8080:8080"
    depends_on:
      - mysql
    environment:
      SPRING_DATASOURCE_URL: jdbc:mysql://mysql:3306/xreport
      SPRING_DATASOURCE_USERNAME: xreport
      SPRING_DATASOURCE_PASSWORD: xreport123

  xreport-frontend:
    image: nginx:alpine
    volumes:
      - ./frontend/dist:/usr/share/nginx/html
      - ./frontend/nginx.conf:/etc/nginx/conf.d/default.conf
    ports:
      - "80:80"
    depends_on:
      - xreport-backend

volumes:
  mysql_data:
```

## 常见问题

### 1. 启动报数据库连接失败

检查：
- MySQL 服务是否启动
- 数据库和用户是否已创建
- `application.yml` 中的连接信息是否正确

### 2. 前端构建内存溢出

增加 Node 内存限制：

```bash
export NODE_OPTIONS="--max-old-space-size=4096"
npm run build
```

### 3. Linux 下 PDF 导出中文不显示

安装中文字体：

```bash
# CentOS/RHEL
yum install -y fontconfig
fc-cache -fv

# Ubuntu/Debian
apt-get install -y fontconfig fc-cache
fc-cache -fv
```

### 4. 跨域问题

后端已配置 CORS，如仍有问题检查 Nginx 代理配置是否正确传递了 CORS 头部。
